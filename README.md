# Scriptly 

## How to deploy and run the service

<details>
<summary>Manually install with NPM and all...</summary>

First, clone the repo and go to root of repo

```bash
git clone https://github.com/jatinrathore/Scriptly.git
cd scriptly
npm install
```

```bash
npm run dev
```

</details>
